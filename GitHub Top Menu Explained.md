#  🐞 Issues 
👉 Used to track problems, tasks, or features

🧠 What is an Issue?
- Issue = task / bug / request

#### 📌 Examples of Issues in your project:
- ❌ “Employee not saving properly”
- 🐞 “Attendance not updating”
- ✨ “Add auto checkout feature”
- 📧 “Email not working”

#### 👇 What you can do in Issues:
-  Create new issue → “New issue”
-Assign to team member
-Add labels (bug, feature, urgent)
- Discuss in comments
- Close when done ✔

#### 🔥 Real Example (Your Case)
You can create issue like:
- 👉 Title: Auto checkout not working
